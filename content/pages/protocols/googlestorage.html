Google Storage S3 compatible service with pricing based on usage.

Documentation

Features

Create Buckets
Both US and Europe locations are supported and share files securely with an authenticated URLs.

Website Configuration
To host a static website on Google Cloud Storage, It is possible to define a bucket as a Website Endpoint. The configuration in File → Info (⌘-I) → Distribution allows you to enable website configuration.

Bucket Logging
Configure bucket access logging to get statistics with Qloudstat.
